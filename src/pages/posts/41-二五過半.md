**ClashX META**  
<https://github.com/MetaCubeX/ClashX.Meta>  
現在 MAC 用的 TUN, 之前用 xray (brew install), 最近發現這個好用多了，規則策略都可以在 UI 上操作;  
主要是一套 clash setting 通用~     
* Android -> flash | clashmeta for android(個人感覺 UI 好難用）
* IPHONE -> 自己使用 QuantumultX ，若是用 clash core 似乎可以試試 Stash
* MAC -> ClashX META
* Router -> PASSWALL (本來想使用 clash core，感覺非常難用放棄)
