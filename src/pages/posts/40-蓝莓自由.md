## Interesting Tools

**Kavitareader**   
<https://wiki.kavitareader.com/>  
电子书库，IPHONE BOOKS， ANDROID Moon+, 有时候阅读进度经常不同步，集中管理的书库还是有点用、support docker   
<img src="https://gz-blog-storage-1252787757.cos.ap-guangzhou.myqcloud.com/weekly/2025/06/40interestingtoolskavitareader.jpg?imageMogr2/format/webp" width="800" />

**Beszel**  
<https://beszel.dev>  
集中管理服务器的一个 container, 直接监控 CPU IO etc. 对我来说监控的指标不够  
<img src="https://gz-blog-storage-1252787757.cos.ap-guangzhou.myqcloud.com/weekly/2025/06/40interestingtoolsbeszel.jpg?imageMogr2/format/webp" width="800" />

**WatchYourLAN**  
<https://github.com/aceberg/WatchYourLAN>  
查看局域网中的设备，之前自己用 GOLANG 手撸，发现 MAC LOOKUP 匹配 vendor 非常少，很多找不到;  

这个也存在这个问题，希望以后能用大数据学习解决;  
这个的优点就是 website，还可以 docker run;   

MAC OS 不能用 DOCKER， 因 HOST 没办法在 container 上找回主机 CIDR，LINUX 上是可以使用。  
<img src="https://gz-blog-storage-1252787757.cos.ap-guangzhou.myqcloud.com/weekly/2025/06/40interestingtoolswatchurlan.jpg?imageMogr2/format/webp" width="800" />

**tssh**  
<https://trzsz.github.io/>  
一个支持拖拉文件到 terminal 上传到服务器工具，个人觉得不怎么安全，就没使用，个人开发者还是可以使用的~  
<img src="https://gz-blog-storage-1252787757.cos.ap-guangzhou.myqcloud.com/weekly/2025/06/40interestingtoolstssh.jpg?imageMogr2/format/webp" width="800" />
